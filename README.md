# Heterogeneity_MS
This repository contains the code and data used in the manuscript “Estimating the role of heterogeneities in the first-wave dynamics of COVID-19 in Nigeria” by Christabel James, Jason Matthiopoulos and Daniel T. Haydon. This study develops a methodology to estimate heterogeneities in COVID-19 transmission from under-reported incidence data. We estimate the parameters of a modified Susceptible Infected Recovered model using Bayesian Markov Chain Monte Carlo techniques. The repository provides a transparent and fully reproducible workflow for the analyses presented in the manuscript.
